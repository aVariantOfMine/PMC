# Yalgaar

Yalgaar is a message voting website where users can post thoughts and vote on others' thoughts.

## Features

- Post thoughts
- Vote on thoughts
- Real-time updates
- Admin controls

## Setup

1. Install dependencies: `npm install`
2. Set up the `.env` file with your configuration
3. Run the server: `npm start`

## Tech Stack

- Node.js
- Express.js
- Socket.IO
- HTML/CSS/JavaScript
